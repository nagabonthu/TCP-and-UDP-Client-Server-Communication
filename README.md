# TCP and UDP Socket Programming in C

This repository contains simple client and server programs implemented using socket programming in C on Linux. The project demonstrates basic communication between a client and a server using TCP and UDP protocols as part of learning TCP/IP networking concepts.

## Concepts

* TCP client-server communication
* UDP datagram communication
* Socket creation
* Binding sockets
* Listening and accepting connections (TCP)
* Sending and receiving data
* Basic networking using the TCP/IP model
